# Parse on .NET 4.6

* Parse Plugins for .NET 4.6
* Depends on Firebase

## Requirement

* Unity
* **.NET 4.6**

## Install

```shell
yarn add "umm-projects/parse_dotnet46#^1.0.0"
```

## Usage

* Use this module if your project depends Firebase and setting of `Scripting Runtime Version` is **.NET 4.6**.
